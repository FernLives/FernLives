## 🌙 About

I'm a **design-first engineer** building intentional web experiences. I don't code for function — I design for feeling.

Currently focused on **[YŪGEN](https://yugen.up.railway.app)**, a community resource platform for Star Citizen. Everything I make prioritizes aesthetic coherence, thoughtful interaction, and restraint.

**Stack of choice:** React, Tailwind, design systems, minimalist architecture.

## 🎨 Current Work

### **YŪGEN STUDIO** — _v2.0_
A civilian intelligence network for Star Citizen players. Resource matrices, refinery data, player profiles, community submissions (moderated).

**What makes it:**
- **Design-led** — Every pixel considered. Gold accents, dark minimal interface, intentional typography.
- **Functional beauty** — The refinery matrix isn't just data; it's readable, scannable, immersive.
- **Community-ready** — User profiles, submission workflows, admin controls. Built for scale from day one.

**[→ Live](https://yugen.up.railway.app/info) · [→ Repository](https://github.com/FernLives/yugen)**
